## Usage
Using Packer (or adapt for your package manager of choice)
```
use 'MetriC-DT/balance-theme.nvim'
```

And somewhere in your `init.vim`:
```vim
colorscheme balance
```
Alternatively, in lua:
```lua
require'balance'.setup()
```
